valid module

valid.py:

En este modulo se realiza la validacion de campos de entrada, según el patrón regex indicado.

class valid.Validation[fuente]

Bases: object

valid_numero(numero)[fuente]

Valida que el campo “ID Paciente” (DNI) contenga solo números.

Patrón:

“^[d]+$”

valid_text(nombre)[fuente]

Valida que los campos “Nombre y Apellido” y “Cirujano” contenga solo letras en mayúscula y minúscula, sin caracteres especiales. Se permiten acentos, espacios y comas.

Patrón:

“^[a-zA-Z áéíóú,]+$”
